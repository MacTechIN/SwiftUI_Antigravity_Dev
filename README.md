# MyFirstAntigravityApp 🚀

AI 어시스턴트 '아론(Aaron)'과 함께 만드는 첫 번째 네이티브 iOS 애플리케이션 프로젝트입니다.

## 🎯 프로젝트 목적
이 프로젝트의 주요 목적은 AI 코딩 어시스턴트와 개발자가 실시간으로 협업하여 앱 스토어 배포까지 이르는 전체 개발 프로세스를 검증하고 학습하는 것입니다. SwiftUI를 기반으로 현대적인 iOS 앱 개발 방법론을 따릅니다.
